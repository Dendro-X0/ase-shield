```markdown
## Anti-SE Shield v1.0.0-beta.2

First **GitHub Release** for the [ase-shield](https://github.com/Dendro-X0/ase-shield) monorepo. Local-first scam protection for freelancers and B2B professionals — extension (Phase 1) + Windows companion (Phase 2).

### Downloads

| Asset | File | Install |
|-------|------|---------|
| **Extension** | `anti-se-shield-extension-1.0.0-beta.2.zip` | Unzip → Chrome/Edge → Developer mode → Load unpacked |
| **Companion** | `Anti-SE Companion_*.exe` (NSIS) | Run installer → tray icon → dashboard opens on first launch |

**Privacy policy:** https://dendro-x0.github.io/ase-shield/privacy.html

> Companion builds are **unsigned** in this release. Windows SmartScreen may warn — choose “More info” → Run anyway for beta testing. Authenticode signing planned for a later release.

### What's new since beta.1

- Renamed repo to **ase-shield** with canonical URLs in `@ase/core`
- **Unified CI** on every push to `main`: lint, typecheck, test, extension zip, NSIS installer, GitHub Pages privacy deploy
- **First-run UX**: companion opens dashboard welcome, 5-step onboarding, popup setup checklist, connection troubleshooting
- **Dev Lab** regression suite (8 scenarios) + dashboard Dev Lab link
- Fixed CI typecheck (build `@ase/core` / `@ase/rules` before `tsc` on clean checkout)

### Quick start

1. Install **Companion** from this release page
2. Load **Extension** from the zip (store listing coming after dashboard UX polish)
3. Complete onboarding → **Practice** → confirm activity on http://127.0.0.1:47123/

Full beta guide: [docs/BETA.md](https://github.com/Dendro-X0/ase-shield/blob/main/docs/BETA.md)

### Known limitations

- Chrome Web Store / Edge Add-ons **not submitted yet** (screenshots after dashboard UX pass)
- WhatsApp / Telegram: link guard only
- Companion installer unsigned (SmartScreen)

Report issues: https://github.com/Dendro-X0/ase-shield/issues
```
