## Anti-SE Shield v1.0.0-beta.3

Public beta for [ase-shield](https://github.com/Dendro-X0/ase-shield): local-first scam protection for freelancers and B2B professionals — browser extension (Phase 1) + Windows companion (Phase 2).

### Downloads

| Asset | File | Install |
|-------|------|---------|
| **Extension** | `anti-se-shield-extension-1.0.0-beta.3.zip` | Unzip → Chrome/Edge → Developer mode → Load unpacked |
| **Companion** | `Anti-SE Companion_*.exe` (NSIS) | Run installer → tray icon → dashboard opens on first launch |
| **Checksums** | `SHA256SUMS.txt` | Verify downloads before install |

**Privacy policy:** https://dendro-x0.github.io/ase-shield/privacy.html

> Companion builds are **unsigned** in this release. Windows SmartScreen may warn — choose “More info” → Run anyway for beta testing.

### What's new since beta.2

- **Universal scanner** — one content script for all HTTPS pages; site adapters demoted to optional hints
- **Marketplace-only default** — automatic scan limited to freelance/B2B hosts; opt in to all HTTPS sites in settings
- **Dashboard overhaul** — shadcn/ui layout, settings sync, incident export, connection troubleshooting
- **Fiverr support** — inbox API enricher + resilient DOM fallback
- **Selection analyze** — right-click selected text on any page
- **Dev Lab** — new Fiverr off-platform scenario (9 scenarios total)

### Quick start

1. Install **Companion** from this release page
2. Load **Extension** from the zip
3. Complete onboarding → **Practice** → confirm activity on http://127.0.0.1:47123/
4. Open a marketplace inbox (Gmail, LinkedIn, Upwork, Fiverr) and confirm the risk badge

Full beta guide: [docs/BETA.md](https://github.com/Dendro-X0/ase-shield/blob/main/docs/BETA.md)

### Known limitations

- Chrome Web Store / Edge Add-ons submission in progress (broad `https://*/*` permission review)
- Companion installer unsigned (SmartScreen)
- Universal scanner on forums/Discord requires disabling “Freelance & B2B sites only” in settings

Report issues: https://github.com/Dendro-X0/ase-shield/issues
